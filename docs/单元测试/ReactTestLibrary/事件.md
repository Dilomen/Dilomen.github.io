## 事件

事件是用户和界面的一种交互方式

```js
fireEvent(node: HTMLElement, event: Event)

fireEvent(
  getByText(container, 'Submit'),
  new MouseEvent('click', {
    bubbles: true,
    cancelable: true,
  })
)
```

```js
fireEvent[eventName]((node: HTMLElement), (eventProperties: Object));

fireEvent.click(
  getByText(container, 'Submit'),
  {
    bubbles: true,
    cancelable: true,
  }
)
```

## example

点击事件

```js
function Example() {
  const [count, setCount] = useState(456789876);
  return (
    <div>
      <p role="count">{count}</p>
      <button onClick={() => setCount(count + 1)}>addBtn</button>
    </div>
  );
}

test("Test render", async () => {
  render(<Example />);
  const btn = screen.getByText("addBtn");
  screen.getByText('0')
  await fireEvent.click(btn, {});
  // 校验被测试内容相关点击后的结果
  screen.getByText('1')
});
```

改变事件，可以直接通过设置事件的属性值来改变输入框的值

```js
const input = screen.getByLabelText("UserName");
fireEvent.change(input, {
  target: {
    value: "Dilomen",
  },
});
// 校验被测试内容相关点击后的结果
```

键盘事件

```js
fireEvent.keyDown(domNode, { key: "Enter", code: "Enter" });
```

自定义事件

```js
createEvent[eventName];

const myEvent = createEvent.click(node, { button: 2 });
fireEvent(node, myEvent);
```
