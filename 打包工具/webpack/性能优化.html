# cache-loader 缓存

第一次会比正常的慢一点，因为需要把结果放到磁盘中，但是之后的打包都会变得更快，因为有缓存。
只需在开销比较大的 loader 之前添加 cache-loader 即可

rules: [
    {
        ...
        use: ["cache-loader", "ts-loader"]
    }
]
